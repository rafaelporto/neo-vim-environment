# Goto Preview

Configuration in `after/plugin/goto-preview.lua`.

## Purpose

Opens LSP go-to targets (definition, type definition, implementation, references) in floating windows instead of jumping to them. Useful for quickly inspecting a symbol without leaving the current context.

## Keymaps

| Key | Action |
|---|---|
| `<leader>pd` | Preview definition |
| `<leader>pt` | Preview type definition |
| `<leader>pi` | Preview implementation |
| `<leader>pr` | Preview references |
| `<leader>pc` | Close all preview windows |
